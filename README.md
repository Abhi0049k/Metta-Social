## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Abhi0049k/Metta-Social.git
   cd your-repo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run react application:
   ```bash
   npm run dev
   ```
